Γραφήματα σχετικά με αγώνες rally.
Υπάρχουν γραφήματα για αγώνες rally που αφορούν διαφορές ανά ειδική διαδρομή καθώς και στη συνολική κατάταξη του αγώνα. Επίσης διαγράμματα που εμφανίζουν την κατανομή ανά ειδική ανά group.

Methodology
    • Dynamic Normalization: Comparisons are made within machinery classes to ensure meaningful data.
    • Smart Zoom: Chart axes automatically adjust to show high-detail views of close battles.
    • Master Data: Every event is backed by a Master CSV containing stage positions and overall standings evolution.
